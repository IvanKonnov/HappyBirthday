# 🌟 Волшебное поздравление с Днём Рождения

**Универсальная интерактивная веб-открытка** с анимированным космосом, фейерверками и персонализированным поздравлением. Подходит для любого человека — друга, коллеги, родственника или второй половинки.

---

## ✨ Концепция: «Зажги свою звезду»

На экране — тёмное ночное небо с парящими частицами звёздной пыли. В центре расположена потухшая звезда. Имениннику нужно **кликнуть по ней**, чтобы «зажечь» — после этого:

- 🌠 Вспыхивает яркий фейерверк из золотых частиц
- 🪐 Вокруг звезды появляются вращающиеся орбитальные кольца
- ✨ Плавно возникает персонализированное поздравление
- 🔥 Каждый новый клик добавляет ещё больше волшебства
